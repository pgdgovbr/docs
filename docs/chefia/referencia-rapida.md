# Referência Rápida — Chefia Imediata

Consulta rápida para as ações mais comuns da chefia.

## Ações frequentes

| O que fazer | Onde | Caminho |
|---|---|---|
| Ver panorama da equipe | Menu → Equipe | `/equipe` |
| Revisar plano enviado por servidor | Equipe → "Revisar e assinar" | `/equipe/planos-trabalho/<id>/revisar` |
| Ajustar plano antes de assinar | Equipe → plano → "Ajustar" | `/equipe/planos-trabalho/<id>/editar` |
| Avaliar registro de um servidor | Equipe → servidor → período pendente | `/equipe/planos-trabalho/<id>` |
| Responder recurso de servidor | Notificação → período com recurso | `/equipe/planos-trabalho/<id>` |
| Criar plano (caso excepcional) | Equipe → servidor → "Criar Plano" | `/equipe/planos-trabalho/novo` |
| Emitir convocação | Equipe → servidor → "Convocar" | — |
| Ver Plano de Entregas da unidade | Equipe → Plano de Entregas | `/equipe/planos-entregas/<id>` |

## Badges da tela de Equipe

| Badge | O que fazer |
|---|---|
| `Aguardando sua assinatura` | Revisar e assinar o plano enviado pelo servidor |
| `Você ajustou — aguardando servidor` | Aguardar — servidor precisa reassinar a sua versão |
| `Rascunho do servidor` | Servidor ainda está elaborando; aguardar |
| `Em execução` | Plano pactuado; nada a fazer (até a próxima avaliação) |
| `Avaliação pendente` | Avaliar o registro enviado pelo servidor |
| `Recurso aberto` | Responder ao recurso (prazo: 7 dias) |
| `Convocação pendente` | Aguardar o atendimento do servidor |

## Prazos críticos

| Prazo | Tempo |
|---|---|
| Responder a um recurso | 7 dias após o servidor abrir o recurso |

## Notas de avaliação

| Nota | Classificação | Justificativa obrigatória? |
|---|---|---|
| 1 | Excepcional | Sim |
| 2 | Alto desempenho | Não |
| 3 | Adequado | Não |
| 4 | Insuficiente | Sim |
| 5 | Insatisfatório | Sim |

## Convocação presencial

| Regra | Valor |
|---|---|
| Prazo mínimo de antecedência | 5 dias corridos |
| Quem pode receber | Servidores em teletrabalho integral |

## Status de plano de trabalho

| Status | Significa |
|---|---|
| Rascunho do servidor | Servidor elaborando; nada a fazer |
| Aguardando sua assinatura | Você precisa revisar e assinar |
| Rascunho da chefia | Você está elaborando (caso de exceção) |
| Aguardando assinatura do servidor | Você ajustou/criou; servidor precisa reassinar |
| Em execução | Plano pactuado e ativo |
| Concluído | Período encerrado |
| Avaliado | Todos os períodos encerrados |
| Cancelado | Plano cancelado por uma das partes |
